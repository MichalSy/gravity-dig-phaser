# Gravity Dig Phaser

Minimal Phaser + TypeScript web spike for Gravity Dig.

## Development

```bash
npm install
npm run dev
npm run build
```

## Deployment

GitHub Actions builds a Docker image to GHCR:

`ghcr.io/michalsy/gravity-dig-phaser.aikogame`

GitOps/ArgoCD deploy target:

`https://gravity-dig-phaser.sytko.de`
