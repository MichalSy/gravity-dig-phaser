## 🌍 Terra-Prime (Tutorial-Planet)

**Schwierigkeit:** ⭐ Einfach  
**Empfohlen für:** Erstes Spiel, Lernen

### Eigenschaften
| Attribut | Wert |
|----------|------|
| Gravitation | 1x (Normal) |
| Grav-Änderung | Nie |
| Core-Stärke | Schwach |
| Anziehung | Minimal |

### Material-Verteilung
- 50% Erde, Sand
- 30% Stein
- 15% Eisen, Kupfer
- 5% Kristalle

### Besondere Blöcke
- Mondstein (flach)
- Keine Gefahren-Blöcke

### Empfohlene Strategie
- Basis-Mechaniken lernen
- Erste Credits sammeln
- Keine Magnet-Stiefel nötig

### Belohnung
- Sichere Einführung
- Stabile Erfahrung
