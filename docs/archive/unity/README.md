# 🎮 Gravity Dig Unity

Gravity Dig - migrated from Godot to Unity 6

## About

An interplanetary digging game with gravity-flip mechanics. Dig deep, flip gravity, find relics.

**Original:** Godot 4.6 + GDScript  
**Current:** Unity 6 LTS + C#

## Deployment

- **URL:** https://gravity-dig-unity.sytko.de
- **Platform:** WebGL
- **Hosting:** k3s cluster with ArgoCD
