# 🌍 GRAVITY DIG

> *Dig deep. Flip gravity. Discover the unknown.*

Ein **intergalaktisches Gravitation-Buddel-Spiel** — jeder Planet hat eigene Physik, und du kannst sie VERÄNDERN.

## 🎮 Das Besondere

- **Gravity-Flip:** Wechsle zwischen Fallen, Steigen und Schwerelosigkeit
- **7 Planeten:** Jeder mit einzigartiger Gravitation (3x Schwere, -1x Invertiert, Chaos-Modus...)
- **Slot-Machine:** Vor jedem Trip ein Glücksspiel für Boni oder Risiken
- **Aktive Artefakte:** Zeitlupe, Phasen-Shift, Mass-Veränderung
- **Gravitations-Rätsel:** Die Physik IST das Rätsel
